# Endless Tower 2

**A momentum roguelite.** Icy Tower's power fantasy — speed becomes height,
walls redirect momentum, combos celebrate mastery — restructured as a run-based
climb: choose your path on a branching map, stack relics into a visibly powerful
build, and take it through boss duels fought entirely with movement.

> Early development. Successor to [endless-tower](https://github.com/TomNeyland/endless-tower),
> rebuilt from scratch with Phaser 4.

## Quick start

```bash
npm install
npm run dev     # http://localhost:8080
```

## Tech

- [Phaser 4](https://phaser.io) + [Vite](https://vite.dev) + TypeScript (strict)
- Art & sounds: [Kenney New Platformer Pack](https://kenney.nl) (CC0 — thanks, Kenney!)
